📝 Modern Todo App with React & Redux
This is a robust and responsive Todo Application built using React.js and Redux for state management. It allows users to manage their daily tasks efficiently while featuring a professional contact form and a modern UI.

🚀 Features
Task Management: Add, complete, and delete todos with real-time updates.

Global State Management: Uses Redux to handle application data across different components.

Professional Contact Form: Implemented with useState and object handling for a seamless user experience.

Conditional Rendering: Includes a toggle feature to show/hide submitted messages.

Responsive Footer: A modern footer with social media icons and hover effects.

Clean Architecture: Built using CSS Modules to ensure scoped styling and avoid class name conflicts.

🛠 Technology Stack
Frontend: React.js (Hooks, Functional Components)

State Management: Redux (Actions, Reducers, Dispatch, useSelector)

Icons: React Icons (fa6, tb)

Styling: CSS Modules

Navigation: React Router DOM
